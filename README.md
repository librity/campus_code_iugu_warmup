# Study Diary CLI

Iugu 4 Devs Warmup: a simple CLI program that helps you manage your studies.

## Instructions

Clone this repo and run `study_diary.rb`:

```bash
$ git clone https://github.com/librity/campus_code_iugu_warmup
$ cd campus_code_iugu_warmup
$ ./study_diary.rb
```
